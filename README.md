## 🛠️ How to Deploy on GitHub Pages (Step by Step)

### Step 1 — Create a GitHub Account
If you don't have one: [github.com/signup](https://github.com/signup)

### Step 2 — Create a New Repository
1. Go to [github.com/new](https://github.com/new)
2. Repository name: `omar-ikram-portfolio`
3. Set to **Public**
4. Click **Create repository**

### Step 3 — Upload Files
1. Click **"uploading an existing file"** on the repo page
2. Drag and drop:
   - `index.html`
   - `Omar_Ikram_CV.pdf` ← **important for the download button**
3. Click **Commit changes**

### Step 4 — Enable GitHub Pages
1. Go to **Settings** tab of the repository
2. Scroll to **Pages** in the left sidebar
3. Under **Source**, select **Deploy from a branch**
4. Branch: `main` | Folder: `/ (root)`
5. Click **Save**

### Step 5 — Your site is live!
Wait ~2 minutes, then visit:
`https://[your-github-username].github.io/omar-ikram-portfolio`
